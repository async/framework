# API Reference

The symbol-level API reference is published through jsdocs.io from the npm package declarations.

## Open API docs

[Open the @async/framework API reference](https://www.jsdocs.io/package/@async/framework).

## Package metadata

The package publishes declaration files for the root entry and public subpaths:

- `@async/framework`
- `@async/framework/browser`
- `@async/framework/server`
- `@async/framework/jsx`
- `@async/framework/vite`
- `@async/framework/runtime`
- `@async/framework/runtime/signals`
- `@async/framework/runtime/events`

## Local docs scope

These GitHub Pages docs explain concepts, workflows, and examples. jsdocs.io owns the symbol browser so the local site does not duplicate API pages.
