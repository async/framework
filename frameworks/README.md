# Framework versions

Each folder in this directory represents a framework package.

- `current` is the canonical package `@async/framework` (currently mapped to v1).
- `v0` contains the v0 implementation package (`@async/framework-v0`).
- `v1` contains the v1 implementation package (`@async/framework-v1`).

These packages are intentionally version-scoped so examples and benchmarks can target a specific framework version directly.
